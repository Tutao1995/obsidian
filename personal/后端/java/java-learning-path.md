# Java 学习路径  
  
## 学习目标  
一句话概括：从语法入门到能独立完成 Java 后端项目。  
  
| 阶段                | 目标                    | 产出           |     |
| ----------------- | --------------------- | ------------ | --- |
| 第一阶段：Java 基础语法    | 能读懂并编写简单 Java 程序      | 控制台练习程序      |     |
| 第二阶段：面向对象编程       | 建立类、对象、封装、继承、多态思维     | 小型对象建模练习     |     |
| 第三阶段：核心 API 与常用能力 | 熟练使用集合、字符串、时间、异常、IO   | 工具类练习与文件处理程序 |     |
| 第四阶段：进阶语法与工程能力    | 理解泛型、反射、注解、多线程、网络     | 综合练习项目       |     |
| 第五阶段：数据库与后端开发     | 能写数据库程序和基础 Web 应用     | CRUD 项目      |     |
| 第六阶段：框架与项目实战      | 掌握 Spring 生态并完成完整后端项目 | 完整后端项目       |     |
| 第七阶段：源码、性能与面试提升   | 强化底层理解与工程思维           | 进阶总结与专题复盘    |     |
  
## 第一阶段：Java 基础语法  
一句话概括：先把语法、执行流程、基本数据操作打牢。  
  
### 1. 开发环境与运行机制  
| 学习点 | 细节 |  
|---|---|  
| JDK / JRE / JVM 区别 | JDK 是开发工具包，JRE 是运行环境，JVM 负责执行字节码 |  
| 编译与运行流程 | `.java` 通过 `javac` 编译成 `.class`，再由 `java` 运行类名 |  
| PATH 与环境变量 | 理解为什么命令行可以直接执行 `javac` 和 `java` |  
| 第一个程序 | `public class`、`main` 方法、文件名与类名一致 |  
| 常见报错 | 类名不匹配、找不到主类、编码问题、路径错误 |  
  
### 2. 基本语法  
| 学习点 | 细节 |  
|---|---|  
| 标识符与关键字 | 命名规范、大小写敏感、保留字不能使用 |  
| 注释 | 单行、多行、文档注释，注释只说明必要背景 |  
| 变量与常量 | 局部变量、成员变量、`final` 常量 |  
| 数据类型 | `byte`、`short`、`int`、`long`、`float`、`double`、`char`、`boolean` |  
| 字面量 | 整数、小数、字符、字符串、布尔值、十六进制 |  
| 类型转换 | 自动类型提升、强制类型转换、精度丢失、溢出问题 |  
  
### 3. 运算符与表达式  
| 学习点 | 细节 |  
|---|---|  
| 算术运算符 | `+ - * / %`，整数除法与浮点除法区别 |  
| 自增自减 | `i++` 和 `++i` 的执行差异 |  
| 比较运算符 | 返回 `boolean`，理解条件判断依赖它 |  
| 逻辑运算符 | `&&`、`||`、`!`，短路特性必须理解 || 赋值与复合赋值 | `=`、`+=`、`-=`，隐式转换注意点 |  
| 三元运算符 | 简单条件分支替代 `if-else` |  
  
### 4. 流程控制  
| 学习点 | 细节 |  
|---|---|  
| `if / else if / else` | 条件顺序、嵌套判断、边界条件 |  
| `switch` | `case`、`break`、穿透现象、适用场景 |  
| `for` 循环 | 计数循环、循环变量作用域 |  
| `while` / `do while` | 条件前置与后置区别 |  
| `break` / `continue` | 跳出循环、跳过本次迭代 |  
| 循环嵌套 | 打印图形、乘法表、遍历二维结构 |  
  
### 5. 数组  
| 学习点 | 细节 |  
|---|---|  
| 一维数组 | 定义、初始化、遍历、默认值 |  
| 二维数组 | 行列概念、本质是数组套数组 |  
| 常见操作 | 求最值、求和、平均值、逆序、查找 |  
| 常见错误 | 下标越界、空指针、长度误判 |  
  
### 6. 第一阶段练习建议  
| 练习题 | 目标 |  
|---|---|  
| 计算器 | 输入两个数和运算符，练习判断与运算 |  
| 猜数字 | 练习循环与条件分支 |  
| 九九乘法表 | 练习嵌套循环 |  
| 数组最大最小值 | 练习遍历与边界处理 |  
| 学生成绩统计 | 练习数组、平均值、分支逻辑 |  
  
## 第二阶段：面向对象编程  
一句话概括：学会用类和对象组织代码，而不是把逻辑全塞进 `main`。  
  
### 1. 类与对象  
| 学习点 | 细节 |  
|---|---|  
| 类的定义 | 属性、方法、构造器的基本结构 |  
| 对象创建 | `new` 的含义、引用变量、本质是对象地址 |  
| 成员变量与局部变量 | 生命周期、默认值、作用域 |  
| 方法定义 | 返回值、参数、调用关系 |  
| 方法重载 | 同名不同参数列表 |  
  
### 2. 封装  
| 学习点 | 细节 |  
|---|---|  
| `private` | 隐藏内部实现 |  
| `public getter/setter` | 通过方法控制访问 |  
| 校验逻辑 | 在 setter 中处理非法值 |  
| 封装目的 | 降低耦合、保护数据、控制对象状态 |  
  
### 3. 构造器与 `this`  
| 学习点 | 细节 |  
|---|---|  
| 无参构造与有参构造 | 初始化对象常用方式 |  
| 构造器重载 | 多种初始化路径 |  
| `this` | 区分成员变量与局部变量、调用本类构造器 |  
  
### 4. 继承与多态  
| 学习点 | 细节 |  
|---|---|  
| `extends` | 子类继承父类成员 |  
| 方法重写 | 返回值、访问权限、参数列表要求 |  
| `super` | 访问父类成员、调用父类构造器 |  
| 多态 | 父类引用指向子类对象 |  
| 向上转型 / 向下转型 | `instanceof` 判断后再转型 |  
| 多态价值 | 解耦调用方与实现类 |  
  
### 5. 修饰符与关键字  
| 学习点 | 细节 |  
|---|---|  
| `static` | 类变量、类方法、静态代码块 |  
| `final` | 修饰类、方法、变量 |  
| `abstract` | 抽象类、抽象方法 |  
| `interface` | 接口定义、实现、面向接口编程 |  
  
### 6. 内部类与枚举基础  
| 学习点 | 细节 |  
|---|---|  
| 成员内部类 | 访问外部类成员 |  
| 匿名内部类 | 快速实现接口或抽象类 |  
| `enum` | 固定常量集合建模 |  
  
### 7. 第二阶段练习建议  
| 练习题 | 目标 |  
|---|---|  
| 学生管理系统 | 类、对象、数组/集合、增删改查 |  
| 图书管理系统 | 封装、构造器、对象数组 |  
| 员工与经理继承模型 | 继承、多态、方法重写 |  
| 支付接口模拟 | 接口、多实现类、多态调用 |  
  
## 第三阶段：核心 API 与常用能力  
一句话概括：把常见标准库用熟，开始写更像真实项目的程序。  
  
### 1. String 与常用类  
| 学习点 | 细节 |  
|---|---|  
| `String` 不可变 | 字符串拼接成本与对象创建 |  
| 常用方法 | `length`、`substring`、`equals`、`contains`、`split`、`replace` |  
| `StringBuilder` | 高频拼接时替代 `String` |  
| 包装类 | `Integer`、`Double`、自动装箱拆箱 |  
| `Math` | 常见数学运算 |  
  
### 2. 集合框架  
| 学习点 | 细节 |  
|---|---|  
| `Collection` 与 `Map` 区别 | 单值集合 vs 键值对集合 |  
| `List` | `ArrayList`、`LinkedList` 使用场景 |  
| `Set` | `HashSet` 去重特性 |  
| `Map` | `HashMap` 存储键值对 |  
| 遍历方式 | for、增强 for、迭代器、entrySet |  
| 集合选择 | 查找、插入、去重、顺序要求 |  
  
### 3. 泛型  
| 学习点 | 细节 |  
|---|---|  
| 泛型类与泛型方法 | 提升类型安全 |  
| 通配符 | `?`、`? extends T`、`? super T` |  
| 为什么需要泛型 | 避免强制类型转换与运行期错误 |  
  
### 4. 异常机制  
| 学习点 | 细节 |  
|---|---|  
| 异常体系 | `Throwable`、`Exception`、`RuntimeException` |  
| `try-catch-finally` | 捕获与资源释放 |  
| `throw` 与 `throws` | 主动抛出、声明抛出 |  
| 自定义异常 | 业务场景下表达明确错误 |  
| 异常处理原则 | 能处理就处理，不能处理就继续抛出 |  
  
### 5. IO 与文件操作  
| 学习点 | 细节 |  
|---|---|  
| File 基础 | 文件、目录、路径 |  
| 字节流 | `InputStream`、`OutputStream` |  
| 字符流 | `Reader`、`Writer` |  
| 缓冲流 | 提高读写效率 |  
| 常见场景 | 读取文本、写日志、复制文件 |  
| 编码问题 | UTF-8、乱码产生原因 |  
  
### 6. 日期时间 API  
| 学习点 | 细节 |  
|---|---|  
| 旧版日期类 | `Date`、`Calendar` 了解即可 |  
| 新版日期类 | `LocalDate`、`LocalTime`、`LocalDateTime` |  
| 格式化 | `DateTimeFormatter` |  
| 时间计算 | 加减天数、比较时间、解析字符串 |  
  
### 7. 第三阶段练习建议  
| 练习题 | 目标 |  
|---|---|  
| 用户登录校验 | `String`、分支、边界处理 |  
| 文本文件统计程序 | IO、字符串处理 |  
| 通讯录系统 | 集合、对象、查找与修改 |  
| 异常版学生管理系统 | 异常机制、输入校验 |  
  
## 第四阶段：进阶语法与工程能力  
一句话概括：开始接触 Java 进阶能力，为后端和面试打基础。  
  
### 1. 多线程与并发基础  
| 学习点 | 细节 |  
|---|---|  
| 线程创建方式 | `Thread`、`Runnable`、`Callable` |  
| 线程安全问题 | 共享变量、竞态条件 |  
| 同步机制 | `synchronized`、锁对象、同步代码块 |  
| 常见类 | `ReentrantLock`、`CountDownLatch`、线程池 |  
| 线程池 | 为什么要池化、核心参数概念 |  
| 并发容器 | `ConcurrentHashMap` 基础认识 |  
  
### 2. 反射与注解  
| 学习点 | 细节 |  
|---|---|  
| Class 对象 | 获取类结构信息 |  
| 反射调用 | 获取构造器、方法、字段 |  
| 注解定义与使用 | 元注解、运行时注解 |  
| 实际用途 | 框架扫描、依赖注入、对象映射 |  
  
### 3. Lambda 与函数式编程  
| 学习点 | 细节 |  
|---|---|  
| 函数式接口 | `@FunctionalInterface` |  
| Lambda 表达式 | 简化匿名内部类 |  
| 方法引用 | `对象::方法名`、`类名::静态方法` |  
| Stream API | `filter`、`map`、`collect`、`forEach` |  
| 使用原则 | 简单清晰时用，复杂逻辑不要硬塞进链式调用 |  
  
### 4. 网络编程基础  
| 学习点 | 细节 |  
|---|---|  
| TCP / UDP 概念 | 面向连接与无连接区别 |  
| Socket 基础 | 客户端与服务端通信 |  
| HTTP 基础理解 | 请求方法、状态码、请求头 |  
  
### 5. 构建工具与项目结构  
| 学习点 | 细节 |  
|---|---|  
| Maven | 依赖管理、生命周期、常见命令 |  
| 项目目录结构 | `src/main/java`、`src/test/java` |  
| `pom.xml` | 坐标、依赖、插件 |  
| 单元测试基础 | JUnit 基本使用 |  
  
### 6. 第四阶段练习建议  
| 练习题 | 目标 |  
|---|---|  
| 多线程售票系统 | 同步与线程安全 |  
| Stream 数据统计 | Lambda、集合处理 |  
| Socket 聊天室简化版 | 网络通信 |  
| Maven 管理的工具项目 | 工程化组织代码 |

## 推荐学习顺序  
一句话概括：按“语法 → OOP → API → 工程化 → 后端框架 → 底层原理”推进。  
  
| 顺序 | 学习模块 |  
|---|---|  
| 1 | 基础语法 + 流程控制 + 数组 |  
| 2 | 面向对象 + 封装继承多态 |  
| 3 | String + 集合 + 泛型 + 异常 + IO |  
| 4 | 多线程 + Lambda + Maven + JUnit |  
| 5 | SQL + JDBC + Servlet + MyBatis |  
| 6 | Spring + Spring Boot + 项目实战 |  
| 7 | JVM + 并发原理 + 性能优化 + 面试 |  
  
## 学习方法  
一句话概括：每学一个知识点，必须马上配套代码练习和小项目。  
  
| 方法 | 要求 |  
|---|---|  
| 学语法 | 当天写 5~10 个小练习 |  
| 学 OOP | 至少做 2 个管理系统练习 |  
| 学集合/IO/异常 | 做真实数据处理题 |

| 学框架 | 必须做完整项目，不只看视频 |

## 每个阶段的完成标准  
一句话概括：不要以“看完”为标准，要以“能独立写出来”为标准。  
  
| 阶段 | 完成标准 |  
|---|---|  
| 基础语法 | 能独立写循环、数组、判断程序 |  
| 面向对象 | 能自己设计类并完成小系统 |  
| 核心 API | 能熟练处理字符串、集合、文件 |  
| 工程化 | 能用 Maven 管理项目并写测试 |  
| 数据库与 Web | 能完成登录 + CRUD 项目 |  
| 框架实战 | 能独立写一个 Spring Boot 后端 |  
| 进阶提升 | 能解释常见原理并优化项目 |
